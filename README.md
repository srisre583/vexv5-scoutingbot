# vexv5-scoutingbot
A Discord bot designed to help VEX V5 teams and coaches make smarter decisions during competitions by providing real-time scouting intelligence. Built with Python, it integrates directly with the RobotEvents API to pull live and historical data on any registered VEX V5 team, then passes that data through an AI model to generate human-readable scouting reports, performance summaries, and alliance recommendations.

# What It Does
Instead of manually browsing RobotEvents or spreadsheets during a tournament, you can simply run a command in Discord and instantly get a breakdown of any team — their recent match history, win/loss record, autonomous performance, ranking points, and more. The AI layer interprets the raw data and turns it into actionable insights, so even newer scouts can quickly understand a team's strengths and weaknesses.

# Reason why it is open-sourced
VEX Robotics is a community built on collaboration, innovation, and the idea that students learn best by building on each other's work. Keeping this bot open source reflects those same values. Whether you're a team that wants to self-host it with your own API keys, a developer who wants to add support for your region's competitions, or a student who wants to learn how real-world bots are built and the code is there for you.
